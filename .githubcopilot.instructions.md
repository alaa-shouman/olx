# Copilot Rules: OLX Lebanon Assessment

## Styling Rules (CRITICAL)
- **NO UI LIBRARIES.** Only use `View`, `Text`, `TouchableOpacity`, `FlatList`, and `StyleSheet` from `react-native`.
- Always use **Flexbox** for layouts.
- For spacing, always use:
  - `paddingStart` / `paddingEnd`
  - `marginStart` / `marginEnd`
  - `textAlign: 'left'` (which flips automatically) or logic based on `I18nManager.isRTL`.

## Code Style
- Use **Functional Components** with hooks.
- Use **TypeScript interfaces** for all component props and API responses.
- Stick to the brand colors:
  - Primary (Burnt Orange): `#c46545`
  - Neutral (Dark Grey): `#434343`

## Implementation Focus
- **Home Screen:** Horizontal lists and grids.
- **Search:** Complex filtering logic.
- **Translations:** Use the `t()` function from `useTranslation` for all hardcoded strings.

## Workflow
- Generate boilerplate for TypeScript interfaces based on JSON samples I provide.
- Suggest "Generic and Reusable" components by default.
